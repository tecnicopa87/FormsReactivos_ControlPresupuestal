Conoce las apliaciones de NetCore combinadas con Angular

  • ASP.NET Core and C# for cross-platform server-side code
  • Angular and TypeScript for client-side code
  • Bootstrap for layout and styling

Como fue construida ésta aplicacion:

  • La base es NetCore 2.1. en VisualStudio2017 usando el cli-command then Back to return here.
  • Angular integration. Es un framework potente ideal para proyectos mediandos a sistemas escalables ng serve. It runs in the background automatically, so your client-side resources are dynamically built on demand and the page refreshes when you modify any file.
  • Efficient production builds. incorporando una librería de terceros como @syncfusion,obtenemos mayor riqueza en controles muy útiles para cada requerimiento dotnet publish también existen opciones como angular.material ng build o simplemente con boostrap, ahead-of-time compiled JavaScript files.

The ClientApp subdirectory is a standard Angular CLI application. If you open a command prompt in that directory, se utilizaron migrations para creacion de tablas en un servidor sqlserverng command (e.g., ng test), or use npm to install extra packages into it.
